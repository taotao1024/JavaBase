## 泛型 (参数化类型)genericity
 *java集合有一个缺点 把一个对象丢给集合之后，集合会忘记对象的类型，当再次取出对象后，对象的编译类型变成Object类型的对象。
* 所谓泛型就是在定义类、接口、方法时使用类型参数，这个类型参数将在声明变量、创建对象、调用方法时动态指定。
|     符号    |   含义  |     备注    | 
|     ----   |   ---- |     -----    |
| ？| 未知类型 | 匹配任何类型 |
| ？extend E | 设置通配符上限 | 匹配任何类型 |
| ？super E | 设置通配符下限 | 匹配任何类型 |
